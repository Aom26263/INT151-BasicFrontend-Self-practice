ข้อ 2 ใช้ push เพิ่มสมาชิกเข้าไปที่ array เเละใช้ ใช้ pop เพื่อลบสมาชิกตัวสุดท้ายออกจาก array เเละมีการใช้ for loop ในการโชว์ผลลัพธ์ของค่าใน array 
ข้อ 3 ใช้ spread operator ในการเเยกคำที่เป็น string ออกมาเป็นตัวอักษร เเละมีการลองสร้าง array เปล่า
